# 📊 CryptoCoin Analysis Project

## 📌 Overview

This project focuses on analyzing cryptocurrency market data using web scraping, data analysis, and visualization techniques. It provides insights into price trends, market behavior, and key performance indicators (KPIs) of various cryptocurrencies.

## 🚀 Features

* 🔍 Web scraping of cryptocurrency data
* 📈 Data analysis using Python
* 📊 Interactive Excel dashboard visualization
* 📉 Trend analysis and insights
* 📁 Structured dataset handling

## 🛠️ Technologies Used

* Python (Pandas, NumPy)
* Scrapy (Web Scraping)
* Jupyter Notebook
* Microsoft Excel (Dashboard & KPIs)
* HTML/CSS

## ⚙️ How to Run the Project

1. Clone the repository
   git clone https://github.com/your-username/CryptoCoin-Analysis.git

2. Navigate to the project folder
   cd CryptoCoin-Analysis

3. Install required libraries
   pip install pandas numpy scrapy

4. Run the application
   python app.py

5. Open Dashboard
   Open Crypto_Analysis_Dashboard.xlsx in Excel to view KPIs and charts

## 📊 Key Insights

* Identifies top-performing cryptocurrencies
* Analyzes price fluctuations over time
* Calculates KPIs such as average downfall and growth trends
